# AI Resume Screening Platform 🚀

## 📌 Project Overview
This project is an AI-based Resume Screening Platform built using Python and Streamlit.

It automatically extracts candidate details from resumes and stores them for recruitment analysis.

## ⚙️ Features
- Resume Upload
- Name & Phone Extraction
- Candidate Database
- Streamlit UI

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
